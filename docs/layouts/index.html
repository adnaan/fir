It's a Library

It’s a Go library solely focused on building reactive interfaces with mostly server-rendered HTML. Flexibility to adopt or discard the model layer.

Progressive Enhancement

Abilty to progressively enhance a plain html/template webapp to a reactive webapp with partial page change abilities.

Real-time UI updates

Publish notifications, chat messages or any partial page change over websockets.

CLI

A productive CLI which can generate dabatabse models and html views built on top of entgo

Error Handling

Has helper functions to simplify handling html form and page errors.

Declarative & Reactive JS

Ships with an alpine.js plugin to capture user interactions in a simple yet powerful way.

Nothing crazy tech

It is built on nothing crazy tech: Go, html/template and Alpine.js. It’s just plain old html templates sprinkled with a bit of javascript. Aims for long term maintanaibility by building on top of the standard html/template
